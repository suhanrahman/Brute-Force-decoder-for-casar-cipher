# Brute-Force-decoder-for-casar-cipher
This C++ program implements a brute force decoder for the Caesar cipher, a classic substitution cipher where each letter is shifted by a fixed number of positions in the alphabet.

The program prompts the user for a cipher text, then tries all 26 possible shift keys (0-25) to generate and display potential plaintext decodings. Non-alphabetic characters remain unchanged. It's a simple demonstration of cryptanalysis on monoalphabetic ciphers, useful for educational purposes or quick decoding tasks.
